# SemiProject-한눈에쇼핑

servlet+jsp 쇼핑몰 사이트

# ⏰개발기간

* 22.08.05-22.08.18

# 💡담당 역할 및 구현기능

* 프로젝트 총괄 및 깃허브 형상관리 https://github.com/Silveright/Semi_Project2
* 마이페이지(주문내역 확인, 정보수정, 배송지 등록)
* 리뷰 게시판(계층형 게시판, 댓글 및 대댓글 기능, 검색 및 페이징 처리)
